{{title}}

You can apply a border to any element by specifying: bd-{t/r/b/l}-{1/2/3/4/5}

.bd-1 .bd-t-1 .bd-r-1 .bd-b-1 .bd-l-1
.bd-5 .bd-t-5 .bd-r-5 .bd-b-5 .bd-l-5

Border Colors

You can colorize the border by applying: bdc-{color}

See colors for a full list of color names.

.bdc-primary .bdc-success .bdc-warning .bdc-danger .bdc-info

Removing Borders

You can individually remove borders by applying: bd-{t/r/b/l}-0

.bd-0 .bd-t-0 .bd-r-0 .bd-b-0 .bd-l-0

Border Radius

You can add border-radius to an element by applying: rounded-{top/right/bottom/left/circle/0}

.rounded-0 .rounded .rounded-top .rounded-right .rounded-bottom .rounded-left .rounded-circle
