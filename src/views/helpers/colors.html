Listed below are the variable/class names and colors of every color used color in Bootstrap {{toolkit.bootstrap}} plus any colors you add to the theme.

You can use a color on an element by applying:
  • Background color: bgc-{color}
  • Border color: bdc-{color}
  • Text color: txtc-{color}

Need to add a new color?
  1. Open the src/assets/toolkit/styles/themes/YOURTHEME/_colors.scss file.
  2. Add your color name and value to the $colors object. Typically the color value is a variable specified before the inclusion of the _colors.scss.
  3. Open the src/data/toolkit.yml file.
  4. Update the colors value with the total number of colors in the $colors object.
